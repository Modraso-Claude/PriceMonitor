# Мониторинг цен Wildberries (свои товары + конкуренты)

Раз в день скрипт проверяет цены выбранных товаров на Wildberries и,
если цена изменилась, присылает уведомление в Telegram. История цен
накапливается в `history.json`.

## Как это работает

- **products.json** — список товаров: артикул (`nm_id`), название и тип
  (`own` / `competitor`).
- **price_monitor.py** — получает текущую цену каждого товара и сравнивает
  с последней сохранённой.
- **history.json** — создаётся автоматически, хранит цены по датам.
- **GitHub Actions** — запускает скрипт каждый день в 07:00 по Москве без
  вашего участия и без необходимости держать компьютер включённым.

## Шаг 1. Найдите артикулы (nm_id) товаров

Артикул — это число в ссылке на товар:
`wildberries.ru/catalog/392074718/detail.aspx` → артикул `392074718`.

Впишите свои товары и товары конкурентов в `products.json`.

## Шаг 2. Создайте Telegram-бота для уведомлений

1. В Telegram напишите **@BotFather** → `/newbot` → следуйте инструкциям.
   В конце вы получите **токен** вида `123456:ABC-DEF...`.
2. Напишите своему новому боту любое сообщение (например, "старт").
3. Откройте в браузере:
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   и найдите в ответе `"chat":{"id": ЧИСЛО, ...}` — это ваш `chat_id`.

## Шаг 3. Разместите проект на GitHub

1. Создайте новый **приватный** репозиторий на github.com.
2. Загрузите туда все файлы из этой папки (можно перетащить через веб-интерфейс
   GitHub, кнопка "Add file → Upload files", либо через git).

## Шаг 4. Добавьте секреты в репозиторий

В репозитории: **Settings → Secrets and variables → Actions → New repository secret**

Добавьте два секрета:
- `TELEGRAM_BOT_TOKEN` — токен бота из шага 2
- `TELEGRAM_CHAT_ID` — ваш chat_id из шага 2

## Шаг 5. Проверьте вручную

Вкладка **Actions → Мониторинг цен Wildberries → Run workflow** — запустит
проверку прямо сейчас, не дожидаясь расписания. Дальше всё будет
происходить автоматически каждый день.

## Если сломалось

Публичный эндпоинт `card.wb.ru`, который использует скрипт — это тот же
запрос, что делает сама витрина WB при отображении цены на странице, но
он не документирован официально и WB иногда меняет его формат. Если в
логах Action появляется ошибка "Не удалось извлечь цену":

1. Откройте DevTools в браузере (F12) → вкладка Network → откройте
   страницу товара на wildberries.ru → найдите запрос к `card.wb.ru`.
2. Посмотрите текущую структуру ответа и поправьте функцию `get_price()`
   в `price_monitor.py` под неё (обычно меняется только путь до поля
   с ценой).

## Альтернатива для своих товаров

Если хотите более надёжный источник именно для **своих** товаров —
Wildberries с 2026 года предоставляет официальный платный API
(pay-as-you-go, стоимость минимальна для 20 товаров). Токен получаете в
личном кабинете продавца: suppliers.wildberries.ru → Настройки → Доступ к API.
Могу дописать в скрипт отдельную функцию под официальный API, если такой
токен у вас есть — так данные по своим товарам будут стопроцентно точными.
